## 📖 About

This is a knowledge-testing game based on the **JRM Youth Traditions** document. It challenges youth members to demonstrate their understanding of the traditions and guidelines set by the Ministry.

## 🎮 Game Modes

- **Scenarios** — Real-life situations: pick the right response
- **True or False** — Quick fire: beat the clock
- **Fill the Gap** — Complete the tradition statement

---

## ✨ Features

- **61 questions** strictly based on the JRM Traditions document
- **Bilingual** — English & Shona (toggle any time)
- **Dark / Light mode**
- **Progressive unlock** — complete one section to unlock the next
- **Scoring system** — base points + time bonus + streak bonus
- **Leaderboard** — compete with others
- **Progress tracking** — see which modes you've completed
- **Group Play** — host a game with a 6-character code, friends join on their devices

---

## 🚀 How to Run Locally

No installation required. Simply open `index.html` in any modern browser.

```
double-click index.html
```

Or from terminal:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

## 🛠️ Tech

- Single-file React 18 app (JSX + inline styles)
- No backend, no build step, no dependencies
- Babel standalone for JSX transpilation
- localStorage for progress & leaderboard persistence
